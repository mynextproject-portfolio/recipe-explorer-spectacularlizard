Import Recipes

Upload a JSON file to import multiple recipes at once. This will replace all existing recipes.

Note: The app auto-loads sample-recipes.json on startup. Use this page to import your own recipe collections.

Upload a JSON file containing an array of recipe objects.
Warning: Importing will replace all existing recipes. This action cannot be undone.
Cancel
JSON Format

Your JSON file should contain an array of recipe objects like this:

[
  {
    "id": "recipe-id-1",
    "title": "Recipe Title",
    "description": "Recipe description",
    "ingredients": ["ingredient 1", "ingredient 2"],
    "instructions": "First, do step 1.\n\nThen, do step 2.",
    "tags": ["tag1", "tag2"],
    "difficulty": "Easy",
    "created_at": "2024-01-01T00:00:00",
    "updated_at": "2024-01-01T00:00:00"
  }
]

Note: The id, created_at, and updated_at fields are optional and will be generated automatically if not provided.
